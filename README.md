# Contact Form UI

A sleek contact form with floating labels and smooth
animations — built with pure HTML, CSS & JavaScript.

## ✨ Features
- Floating labels — animate up on focus
- Error shake animation on invalid fields
- Email validation
- Loading spinner on submit
- Success overlay with checkmark
- Reset — send another message
- Glassmorphism card design
- Contact themed background image

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript (vanilla)

## 🚀 How it works
Floating labels use CSS `:focus` and
`:not(:placeholder-shown)` selectors to animate
the label position. Validation checks empty fields
and email format before showing loading state.
Success overlay fades in after 2 second loader.

## 📂 Usage
Open `contact-form.html` directly in your browser.
No setup needed.
